# Obsidianで現在時刻を楽に入力する

私のデイリーノートのフォーマットは以下

```
HH:mm

- 書くこと1
- 書くこと2

HH:mm

- 書くこと3
- 書くこと4
```

上記からも分かる通り現在時刻を頻繁に打つので[[Obsidian]]での入力を楽にしている

- 実現できること
  - `/ + Enter`で`HH:mm`を入力できる
- やっていること
  - 以下のCore Pluginsを有効にする
    - Command palette
    - Slash Commands
    - Templates
  - Templatesの設定でTime Formatを`HH:mm`にする
    - デフォルトでそうなってるかも
  - Command paletteの設定で`Templates: Insert current time`をPinする
